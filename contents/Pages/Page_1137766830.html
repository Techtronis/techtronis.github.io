2 ADEV Operating Environment

The environment described on this page is the information at the time ADEV support was implemented. For the latest operating environment information, see NintendoSDK Documents > NintendoSDK Overview > Operating Environment.

The following environment is required for ADEV setup and application execution.

Operating system
  • Microsoft Windows 10 Pro (64-bit) English or Japanese version
Note

An account with administrator rights is required.

Note

Behavior is not assured if you change the system locale in the operating system to something other than the default value.

.NET Framework
  • Microsoft .NET Framework 4.7.2 or later
Info

If not already installed, download and install from the following link.

https://support.microsoft.com/en-us/help/4054531/microsoft-net-framework-4-7-2-web-installer-for-windows

.NET Core SDK
  • .NET Core SDK 3.1.410
Info

If this is not already installed, use the installer at the following path.

  • NintendoSDK\Installers\DotNetSdk\v3.1\dotnet-sdk-3.1.410-win-x64.exe
NX Add-On
  • NintendoSDK NX Add-On 13.2.0 and later
SystemUpdater
  • NintendoSDK SystemUpdater for NX 13.0.0-3.0 and later

In addition, you need the following environments to execute 8 Building and Running Applications (For Developers).

Visual Studio
  • Microsoft Visual Studio 2017 Professional Edition version 15.9.30
    • English language bundle required
  • Microsoft Visual Studio 2019 Professional Edition version 16.10.3 (For additional considerations when using Visual Studio 2019, see the NintendoSDK Documents.)
    • English language bundle required
Windows SDK
  • Windows SDK 10.0.15063.0 (when using Visual Studio 2017)
  • Windows SDK 10.0.19041.0 (when using Visual Studio 2019)
Info

To use the NintendoSDK, you must install the Visual C++ redistributable packages.
If they are not installed, use one of the following installers to install the packages. (You must install all four of them.)
