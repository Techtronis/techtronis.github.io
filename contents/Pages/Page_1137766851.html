5 Setting Up NintendoSDK

The NintendoSDK is obtained from the Nintendo server using NDI. NDI downloads the NintendoSDK from the Nintendo server, and then extracts and installs the necessary files.

Installing the NintendoSDK Package

Info

You must have Windows Administrator rights to use NDI.

Step Operation
1

Start NDI and select Dev Environments > Add Environment > Create My Own > Install to Disk.

→ →


2

In the Connect to Nintendo Server dialog box, enter the following information.

  • Server: Select Nintendo Developer Portal.
  • Nintendo Developer ID: Enter the NDID you use when logging in to the Nintendo Developer Portal.

Now click Connect to open the Web Credentials dialog box. Enter your password and log in.

Info

NDI communicates with the Nintendo Developer Portal because the packages it handles are based on Nintendo Developer Portal permissions.
3

Select the Platform and Tool Set as shown in the following image, and then click Next.
4

Select the version of the SDK to install and then click Next.
5

Enter the path to the directory where the environment will be created, the name of the directory (environment name), and click Next.

Info

The NintendoSDK is installed in this directory.

To install multiple versions or installation package types of the NintendoSDK, you can create multiple directories.

Info

False Positives From Antivirus Software

When installing various NintendoSDK packages, NintendoSDK files can be detected incorrectly as viruses by antivirus software.
When such false positives occur, exclude the NintendoSDK install destination folder from the virus scan, and try installing again.
6

Select the bundle of packages that best matches your needs and click Next.

If you select Go to Detailed Package Management Instead at the bottom of the screen, you can instead choose which packages to install.
7

Check the type and version of the package to be installed. If there are no issues, click Install. Download and installation begins.


→
8

USB driver installation is sometimes verified during NintendoSDK installation.

Install the USB driver at this time because it will be used in features that will be added in future releases.
9

On the final screen of the NintendoSDK installation, click Finish.


Click OK to start the NintendoSDK management screen that allows options in the installed environment to be listed.
The NintendoSDK management screen is used in later procedures, so leave it running.
