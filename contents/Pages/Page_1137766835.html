4 Preparing Nintendo Dev Interface

About Nintendo Dev Interface

Nintendo Dev Interface (NDI) is a tool for setting up and managing the development environment.

Installing NDI

Info

To install NDI, you must have Windows administrator rights.


Step Operation
1

Download the NDI installer package from Nintendo Developer Portal > DEVELOPER SUPPORT > Nintendo Switch > Downloads.
2

Double-click the installer from the downloaded package to run it.
3

Select the language for installation.
4

Follow the installer instructions to configure the program.
You can change the installation path from this screen.

On this screen, specify shortcut creation and registration to the Start Menu.
If you select Add application directory to PATH, the directory with the NDI program is added to the PATH environment variable.
This option is useful for calling NDI from the command line.

To start the installation, click Next.

Note: NDI requires .NET Framework 4.6.2.
　 If .NET Framework 4.6.2 is not installed, its installer starts.
　 Follow the instructions to install .NET Framework 4.6.2.

5

To finish the installation, click Finish.
Note: If you have installed .NET Framework 4.6.2 using the preceding procedure, restart Windows before you start Nintendo Dev Interface and run it from the Start menu.

A shortcut to the program is added to the Start menu.
