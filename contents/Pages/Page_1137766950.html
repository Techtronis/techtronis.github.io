10 Running Applications (For Non-Developers)

You can start applications installed on the ADEV console from the DevMenu.

Running Installed Applications

Step Operation
1

Turn on the ADEV console.

A screen like the following is displayed when the DevMenu is started.

If this screen is not displayed, from the menu on the left, select Application.

A list of applications installed on the ADEV console is displayed.
2

Tap the application you want to run.

The selected application starts.
